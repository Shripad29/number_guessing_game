# 🎯 Number Guessing Game

An interactive and beginner-friendly web-based Number Guessing Game built using **HTML**, **CSS**, and **JavaScript**.

In this game, the user tries to guess a random number between **1 and 100** within **10 attempts**. The game gives hints like "Too High" or "Too Low" after each guess, and ends with a success or failure message. A new game can be started after completion.

---


## 🚀 Features

- 🎲 Random number generated between 1 and 100
- 🔢 Shows previous guesses
- 🧠 Displays hints whether the guess is too low or too high
- 🧾 Track remaining attempts
- 🔄 Option to restart the game after win/loss
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

- **HTML** – Structure
- **CSS** – Styling & Layout
- **JavaScript** – Game Logic & DOM Manipulation
